## üîß Setup Instructions

### 1. Get Your Groq API Key
1. Visit [console.groq.com](https://console.groq.com)
2. Sign up for a free account
3. Navigate to API Keys section
4. Create a new API key
5. Copy the key (starts with `gsk_`)

### 2. Configure the Application
1. Open `reflectai.html` in your browser
2. When prompted, paste your Groq API key
3. Choose to save the key locally for convenience
4. Start using the application!

### 3. First Steps
1. **Choose Your Mode**: Toggle between Learning and Quick modes
2. **Try Example Problems**: Click on any category card to load sample problems
3. **Start Learning**: Paste your problem or ask a question
4. **Track Progress**: Use the sidebar to view your learning history and statistics

## üéØ How It Works

### Learning Mode (Recommended)
1. **Present Problem**: Share your coding problem, math question, or concept
2. **Reflection Phase**: AI asks you to explain what you understand first
3. **Progressive Hints**: Receive hints at 4 different levels:
   - Level 1: Gentle nudge
   - Level 2: Specific direction  
   - Level 3: Step-by-step guide
   - Level 4: Detailed walkthrough
4. **Final Solution**: Get complete answer when ready

### Quick Mode
- Direct answers for immediate help
- Perfect for quick clarifications or urgent questions
- Still educational but more straightforward
